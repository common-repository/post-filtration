== Description ==

Post Filtration is a simple plugin which shows the total words of a post and thumbnail of the post at All Post section in dashboard. It has a great facilities that you will be able to filter your posts basis on Post Words Number and with and without the presence of Post Thumbnail.

Features Include:

* Show Total Number of Words of each Post.
* Show the Featured Image (thumbnail) of each Post.
* Posts are sortable with Posts Words
* Posts are sortable with and without presence of Post Thumbnail
* It is very light-weight
* No extra settings or configuration is needed.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Go to Plugin Menu from your dashboard.
2. Click on Add New
3. Now upload "post-filtration.zip" file and click install now.
4. Finally active it. It is done.
